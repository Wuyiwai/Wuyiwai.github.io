> 记录top常用使用技巧

<!--more-->

1. 按cpu存排序
```
# linux下
$ top
$ 然后按shift p或者直接大写锁定+P

# mac下
$ top
$ 然后o
$ 然后输入该栏目name,大小写不严格
$ 比如按cpu降序,则是o+CPU
```

2. 按cpu排序
```
# linux下
$ top
$ 然后按shift m或者直接大写锁定+M

# mac下
$ top
$ 然后o
$ 然后输入该栏目name,大小写不严格
$ 比如按cpu降序,则是o+mem
```
